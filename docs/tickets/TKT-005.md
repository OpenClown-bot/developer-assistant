# TKT-005: Define Hermes Runtime Integration Contract

## 1. Scope

Document the v0.1 contract between Hermes Agent runtime behavior and repository governance artifacts.

## 2. Non-scope

- Do not implement the runtime adapter.
- Do not install or run Hermes Agent.
- Do not enable marketplace skills/plugins.
- Do not add credentials or secret configuration.

## 3. Required Context

- `AGENTS.md`
- `CONTRIBUTING.md`
- `docs/architecture/ARCH-001.md`
- `docs/architecture/adr/ADR-001-platform-foundation.md`
- `docs/architecture/adr/ADR-002-repository-state.md`
- `docs/architecture/adr/ADR-003-plugin-supply-chain.md`

## 4. Acceptance Criteria

- [ ] A document defines Hermes runtime inputs: Telegram event, project binding, role, required context paths, task prompt, allowed files, and expected outputs.
- [ ] A document defines Hermes runtime outputs: status, files changed, validation commands, founder questions, blockers, progress report text, and handoff summary.
- [ ] The contract states repository artifacts remain the authoritative governance state.
- [ ] The contract defines what operational state is stored outside the repository.
- [ ] The contract includes security requirements for secrets, skill/plugin use, sandboxing, and dangerous command approval.
- [ ] The contract names OpenClaw only as a later possible gateway/control UI addition.
- [ ] `python scripts/validate_docs.py` passes.

## 5. Allowed Files

- `docs/architecture/`
- `docs/tickets/TKT-005.md` Execution Log only

## 6. Test/Validation Requirements

- Run `python scripts/validate_docs.py`.
- Manually check consistency with ADR-001, ADR-002, and ADR-003.

## 7. PR Requirements

- Link this ticket.
- Include validation output.
- Identify any follow-up tickets needed for actual runtime implementation.

## 8. Risks

- Too much abstraction could delay useful Telegram delivery.
- Too little detail could let Hermes memory/session state become the hidden source of truth.

## 9. Dependencies

- TKT-001 through TKT-004 are done.
- ARCH-001, ADR-001, ADR-002, and ADR-003 are approved.
