# TKT-018: Add Minimal Trial Vehicle Implementation

## 1. Scope

Add a tiny, low-risk implementation target that can be used as the selected ready trial vehicle for `TKT-011@0.2.0`.

The implementation must add one non-runtime test-support helper for sanitized label assertions and direct unittest coverage for that helper. The helper should make future tests easier to read when asserting that committed fixtures or evidence use sanitized labels such as `chat:founder` and `user:founder` instead of raw Telegram identifiers.

This ticket exists to exercise the normal Executor, Reviewer, PR-Agent, CI, and founder-acknowledgement flow with a real but harmless repository improvement. It must be safe to implement without live Telegram, GitHub, VPS, or external-service access.
